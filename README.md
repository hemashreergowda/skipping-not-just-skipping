# skipping-not-just-skipping
we need to skip to the letter, based on the number of alphabets we give as input.

for eg, 
        i/p             ---------->            o/p
         a                  -1                  z
        abc                 -3                 xyz
      dgsfeteg              -8               vykxwlwy
      
input:
1. How many skipping sequences you wanna generate
2. the input alphabets in each string
